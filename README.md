# 🎮 Learn with Fun: Python Quiz Game

A complete **Multiple Choice Quiz Game** created entirely with Python.
This project runs in the command line and challenges users with 20 MCQ questions from different areas such as general knowledge, science, mathematics, programming, and history.

It highlights the practical use of core Python fundamentals in a simple yet structured way.

---

## 🚀 Features

* 📚 20 Predefined Multiple Choice Questions
* 🔁 Strict input validation (accepts only A / B / C / D)
* 🧮 Instant score calculation
* 🏆 Final result with performance-based feedback
* 🧩 Clean, beginner-friendly code structure
* ➕ Easy to expand with more questions

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Save the file as `quiz_game.py`.
3. Open Terminal or Command Prompt in that folder.
4. Run the following command:

```
python quiz_game.py
```

---

## 📌 How It Works

* The `run_quiz()` function stores all questions inside a **list of dictionaries**.

* Each question includes:

  * The question text
  * Four options (A, B, C, D)
  * The correct answer

* The user must enter a valid option (A/B/C/D).

* The program verifies the answer and updates the score.

* After all questions are completed, the total score is displayed along with a performance message based on the result.
